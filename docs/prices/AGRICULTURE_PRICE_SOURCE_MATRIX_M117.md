## Verification Notes

- Verified means the source page or catalog was reachable during M117 research and had enough public information to classify it.
- Needs verification means the source looks promising but needs endpoint, licensing, freshness, or owner-side permission review before implementation.
- Future means useful later but not recommended for V1 connection without additional permissions or a stable official feed.

## Initial Ranking For V1

1. MOC Open Data DIT API for rice and selected public market items.
2. NABC/OAE Farm Plus if an approved API/export path is available.
3. NABC/OAE catalog for monthly historical/farm-gate context.
4. RAOT for rubber if stable export or permission exists.
5. OCSB catalog for sugarcane seasonal/annual context.
6. Owner-curated manual JSON/table for any gap that cannot be safely automated yet.
