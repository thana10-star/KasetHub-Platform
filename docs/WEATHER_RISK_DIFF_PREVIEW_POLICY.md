# Weather Risk Diff Preview Policy

M80 adds local diff previews for future weather risk rule review.

Diff previews cover:

- threshold changes
- wording changes
- disclaimer changes
- blocked action changes
- risk-level mapping changes

Diffs are not persisted and are not reviewed. They exist to show what a future reviewer would need to inspect before any rule activation.

## Required Protection

Diff previews must never remove required disclaimers, weaken blocked actions, add products/sponsors, add chemical/fertilizer doses, or turn broad caution into prescriptive instructions.
