# Offline Article Source Metadata Standard

M69 defines the future source metadata shape for article review.

Each source metadata item should include:

- source title
- source type
- source owner or organization
- reviewed date
- freshness status
- source confidence
- citation placeholder
- field applicability note

## Current Status

All M69 source metadata is placeholder-only. It is useful for review workflow design but not a final citation.

## Rules

- Placeholder sources block publish readiness.
- Soil test or local expert sources are required before risky soil claims.
- Finance, loan, government, seasonal, and rate-related content must use CMS/freshness metadata later.
- Source metadata must not become a sponsor or affiliate slot.
