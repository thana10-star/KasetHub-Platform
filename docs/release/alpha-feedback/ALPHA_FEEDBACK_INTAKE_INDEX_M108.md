## Privacy Reminder

Do not collect:

- real names unless explicitly necessary and consented
- phone numbers
- exact home/farm location
- financial details
- passwords or credentials
- sensitive personal data

Feedback should focus on app usability, route clarity, mobile readability, trust/privacy concerns, and feature usefulness.

## Triage Rules

- Classify every response by category and severity.
- Fix blockers first.
- Fix confusing UX and safety/privacy clarity before adding features.
- Do not add broad new features from the first feedback wave.
- Do not start Android wrapper work until feedback shows the core web/PWA app is useful and no launch blocker remains.
- Keep AI provider, GPS, cloud sync, OCR, receipt upload, notifications, and wrapper work separate.
