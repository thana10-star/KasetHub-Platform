# Store Listing Copy Draft M103

## Purpose

This is a Thai-first draft for KasetHub V1 store listing copy. It should be reviewed by the owner before use and adjusted to match the final wrapper/release channel.

The copy should stay honest: KasetHub helps farmers ask, plan, check, calculate, and record. It must not promise guaranteed correctness, official warnings, profit, yield, or expert replacement.

## App Name Options

- KasetHub
- KasetHub เกษตรผู้ช่วย

Recommendation: use `KasetHub` as the app name if the icon/listing subtitle can carry the Thai context. Use `KasetHub เกษตรผู้ช่วย` if the store audience needs an immediate Thai agriculture signal.

## Short Description Options

1. ถาม AI เช็กอากาศ ใช้เครื่องมือ และบันทึกฟาร์มง่าย ๆ
2. ผู้ช่วยเกษตรในมือถือ สำหรับถาม วางแผน และจดบันทึกฟาร์ม
3. แอพเกษตรสำหรับเช็กอากาศ ถาม AI และใช้เครื่องมือพื้นฐาน

## Long Description Draft

KasetHub คือแอพผู้ช่วยเกษตรสำหรับเกษตรกรไทย ใช้ถามเรื่องเกษตร เช็กอากาศ ใช้เครื่องมือคำนวณพื้นฐาน อ่านความรู้ และจดบันทึกฟาร์มง่าย ๆ ในมือถือเครื่องเดียว

สิ่งที่ทำได้ใน V1:

- ถาม AI เกษตรเรื่องพืช ดิน ปุ๋ย โรค แมลง อากาศ และการจัดการฟาร์ม
- เช็กพยากรณ์อากาศและข้อมูลความเสี่ยงเกษตรเพื่อช่วยวางแผนงานประจำวัน
- ใช้เครื่องมือคำนวณ เช่น ระยะปลูก ปุ๋ย พื้นที่ ผลผลิต และต้นทุนเบื้องต้น
- อ่านความรู้และบทความเกษตรที่เหมาะสำหรับการเริ่มต้น
- ใช้ฟาร์มของฉันและสมุดฟาร์มแบบง่าย เพื่อบันทึกแปลง รายรับรายจ่าย และผลผลิต
- เก็บข้อมูลฟาร์มแบบ local-first บนอุปกรณ์ใน V1 โดยยังไม่เปิด cloud sync

หมายเหตุด้านความปลอดภัย:

คำตอบจาก AI และข้อมูลในแอพเป็นตัวช่วยในการวางแผนเบื้องต้น ผู้ใช้ควรตรวจสอบกับฉลากสินค้า เจ้าหน้าที่เกษตร ผู้เชี่ยวชาญ แหล่งข้อมูลที่น่าเชื่อถือ และสภาพจริงในพื้นที่ก่อนตัดสินใจเรื่องสำคัญ โดยเฉพาะการใช้สารเคมี โรคพืช ความเสียหายจากอากาศ ต้นทุน และรายได้

## Feature Copy Blocks

AI:

- ถามคำถามเกษตรเป็นภาษาไทย
- เหมาะสำหรับถามแนวทางเบื้องต้นเกี่ยวกับพืช ดิน ปุ๋ย โรค แมลง และการดูแลแปลง
- มีข้อความเตือนให้ตรวจสอบข้อมูลสำคัญก่อนนำไปใช้จริง

Weather:

- เช็กอากาศจากข้อมูลพยากรณ์
- ใช้เพื่อช่วยวางแผนงานฟาร์มประจำวัน
- ไม่ใช่ระบบเตือนภัยอากาศอย่างเป็นทางการ

Tools:

- เครื่องมือคำนวณพื้นฐานสำหรับเกษตรกร
- ใช้ช่วยประเมินระยะปลูก ปุ๋ย ผลผลิต และต้นทุนเบื้องต้น
- ผลลัพธ์เป็นค่าประมาณและควรตรวจสอบกับเงื่อนไขจริง

My Farm:

- บันทึกข้อมูลฟาร์มแบบง่าย
- เริ่มจากแปลง รายรับรายจ่าย และผลผลิต
- ข้อมูล V1 อยู่บนอุปกรณ์เป็นหลัก ยังไม่เปิด cloud sync

## Do Not Claim

Do not use these claims in the store listing:

- AI always correct.
- AI replaces experts, officers, agronomists, or doctors.
- KasetHub is an official government tool.
- KasetHub guarantees crop profit, cost reduction, yield, harvest, or disease recovery.
- Weather is an official emergency/weather warning system.
- Pesticide, fertilizer, or chemical dosage is certain for every field.
- Farm Records are official accounting, tax, loan, or legal records.
