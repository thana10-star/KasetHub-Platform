## Long Description Draft

KasetHub คือแอพผู้ช่วยเกษตรสำหรับเกษตรกรไทย ใช้ถามคำถามเกษตร เช็กอากาศ ใช้เครื่องมือคำนวณพื้นฐาน อ่านความรู้ และจดบันทึกฟาร์มง่าย ๆ ในมือถือเครื่องเดียว

สิ่งที่ทำได้ใน V1:

- ถาม AI เกษตรเรื่องพืช ดิน ปุ๋ย โรค แมลง อากาศ และการจัดการฟาร์ม
- เช็กพยากรณ์อากาศและข้อมูลความเสี่ยงเกษตรเพื่อช่วยวางแผนงานประจำวัน
- ใช้เครื่องมือคำนวณ เช่น ระยะปลูก ปุ๋ย พื้นที่ ผลผลิต และต้นทุนเบื้องต้น
- อ่านความรู้และบทความเกษตรที่เหมาะสำหรับการเริ่มต้น
- ใช้ฟาร์มของฉันและสมุดฟาร์มแบบง่าย เพื่อบันทึกแปลง รายรับ รายจ่าย และผลผลิต
- เก็บข้อมูลฟาร์มแบบ local-first บนอุปกรณ์ใน V1 โดยยังไม่เปิด cloud sync

หมายเหตุด้านความปลอดภัย:

คำตอบจาก AI และข้อมูลในแอพเป็นตัวช่วยในการวางแผนเบื้องต้น ผู้ใช้ควรตรวจสอบกับฉลากสินค้า เจ้าหน้าที่เกษตร ผู้เชี่ยวชาญ แหล่งข้อมูลที่เชื่อถือได้ และสภาพจริงในพื้นที่ก่อนตัดสินใจเรื่องสำคัญ โดยเฉพาะการใช้สารเคมี โรคพืช ความเสียหายจากอากาศ ต้นทุน และรายได้
