# Final Owner Actions M106

## Purpose

This is the final owner checklist before sharing the V1 Cloudflare/PWA preview with testers or the owner's YouTube audience.

## Required Before Wider V1 Feedback

- [ ] Choose support contact.
- [ ] Publish privacy page.
- [ ] Record privacy policy URL.
- [ ] Approve final app icon concept.
- [ ] Generate/export final icon files.
- [ ] Capture final screenshots.
- [ ] Approve final screenshots.
- [ ] Confirm production URL.
- [ ] Confirm production Weather still loads live Open-Meteo data.
- [ ] Confirm `/app/weather` still has no GPS prompt.
- [ ] Confirm store/listing copy does not overclaim AI, Weather, yield, profit, or official status.
- [ ] Share V1 preview with selected test users or YouTube audience.
- [ ] Collect feedback.
- [ ] Decide Android wrapper timing after first feedback round.

## Recommended Feedback Questions

- Is the first screen clear?
- Can users find Ask AI?
- Is Weather readable and useful?
- Are tools easy to understand?
- Is My Farm/Farm Records simple enough?
- Did any page feel unfinished or confusing?
- Did users expect GPS, login, cloud sync, or notifications?

## Stop Conditions

Do not move to Android wrapper if:

- privacy/support URL is still missing
- final icon is not approved
- screenshots are not approved
- production Weather is broken
- users report a blocking mobile layout issue
- copy implies official, guaranteed, or expert-replacement results
