# Alpha Tester Route Guide M107

## คำแนะนำก่อนเริ่ม

กรุณาอย่าใส่ข้อมูลส่วนตัวสำคัญ เช่น ชื่อจริง เบอร์โทร ที่อยู่ รายได้จริง รหัสผ่าน หรือข้อมูลการเงินในแอพหรือแบบฟอร์ม feedback

KasetHub รุ่นนี้เป็น alpha preview เพื่อขอ feedback การใช้งานเบื้องต้น ยังไม่ใช่รุ่นสมบูรณ์ และไม่ใช่คำแนะนำจากหน่วยงานรัฐหรือผู้เชี่ยวชาญแทนการตรวจสอบจริง

## 1. เปิดหน้าแรก

Route: `/app`

ลองทำ:

- ดูว่าหน้าแรกเข้าใจง่ายไหม
- หาปุ่มหรือทางเข้า Ask AI
- ดูว่า Weather, Tools, My Farm หาเจอไหม

Feedback ที่อยากได้:

- หน้าแรกบอกชัดไหมว่าแอพช่วยอะไร
- ตัวหนังสืออ่านง่ายไหม
- ปุ่มหรือเมนูแตะง่ายไหม

## 2. ลองถาม AI

Route: `/app/ai`

ลองทำ:

- อ่านตัวอย่างคำถาม
- ลองพิมพ์คำถามเกษตรทั่วไปที่ไม่ใส่ข้อมูลส่วนตัว
- อ่านคำเตือนด้านความปลอดภัย

Feedback ที่อยากได้:

- หน้า AI เข้าใจง่ายไหม
- คำตอบหรือข้อความช่วยให้วางแผนได้ไหม
- มีส่วนไหนที่ทำให้เข้าใจผิดว่า AI ถูกเสมอหรือไม่

คำเตือน:

- อย่าใส่ข้อมูลส่วนตัวสำคัญ
- ตรวจสอบคำแนะนำสำคัญกับฉลากสินค้า เจ้าหน้าที่เกษตร หรือแหล่งข้อมูลที่เชื่อถือได้

## 3. เช็กอากาศ

Route: `/app/weather`

ลองทำ:

- ดูสภาพอากาศปัจจุบัน
- ดูฝน อุณหภูมิ ลม หรือความเสี่ยงเกษตร
- เช็กว่าอ่านบนมือถือได้ง่ายไหม

Feedback ที่อยากได้:

- ข้อมูลอากาศพอใช้วางแผนงานฟาร์มเบื้องต้นไหม
- สีความเสี่ยงเข้าใจง่ายไหม
- มีข้อมูลไหนรกหรือเข้าใจยากไหม

คำเตือน:

- Weather ใช้ตำแหน่งแบบกว้าง ไม่ใช่ GPS
- ไม่ใช่ระบบเตือนภัยอากาศอย่างเป็นทางการ

## 4. ลองใช้เครื่องมือคำนวณ

Route: `/app/calculators`

ลองทำ:

- เปิดรายการเครื่องมือ
- ลองเครื่องมือที่เกี่ยวกับระยะปลูกหรือปุ๋ย
- ดูว่าช่องกรอกและผลลัพธ์อ่านง่ายไหม

Feedback ที่อยากได้:

- เครื่องมือไหนน่าใช้ที่สุด
- ช่องกรอกยากไหม
- ผลลัพธ์เข้าใจง่ายไหม

คำเตือน:

- ผลลัพธ์เป็นค่าประมาณเพื่อช่วยวางแผน
- ไม่ใช่คำรับรองผลผลิต กำไร หรือปริมาณสารเคมีที่ถูกต้องเสมอ

## 5. เปิดฟาร์มของฉัน

Route: `/app/my-farm`

ลองทำ:

- ดูภาพรวม My Farm
- หา entry ไปสมุดฟาร์มหรือบันทึกต่าง ๆ
- ดูว่าหน้าแน่นเกินไปไหม

Feedback ที่อยากได้:

- My Farm เข้าใจง่ายไหม
- ควรมีอะไรบนหน้าแรกของ My Farm เพิ่มหรือลดไหม
- ข้อความ local-first หรือ sync-off ทำให้สับสนไหม

คำเตือน:

- อย่าใส่ข้อมูลฟาร์มจริงที่เป็นส่วนตัวถ้ายังไม่พร้อม
- V1 ยังไม่มี cloud sync

## 6. ลองสมุดฟาร์มแบบง่าย

Route: `/app/farm-records`

ลองทำ:

- ดู Basic Mode
- ลองดูปุ่มหลัก เช่น เพิ่มแปลง รายรับรายจ่าย หรือผลผลิต
- เช็กว่าขั้นตอนดูง่ายไหม

Feedback ที่อยากได้:

- ปุ่มหลักพอเข้าใจไหม
- ควรลดคำหรือเพิ่มคำอธิบายตรงไหน
- ตัวหนังสือและช่องกรอกเหมาะกับมือถือไหม

คำเตือน:

- ข้อมูล V1 เก็บในเครื่อง/เบราว์เซอร์เป็นหลัก
- ไม่มี cloud backup ใน alpha นี้

## 7. เปิดโปรไฟล์ / วิธีใช้

Routes:

- `/app/profile`
- `/app/help`

ลองทำ:

- อ่านวิธีเริ่มใช้
- ดูว่าหาที่ติดต่อ support/privacy ได้ไหม
- ดูว่าการตั้งค่าเข้าใจง่ายไหม

Feedback ที่อยากได้:

- หน้าวิธีใช้ช่วยให้เริ่มได้จริงไหม
- มีคำไหนดู technical หรือเข้าใจยากไหม
- ต้องการช่องทาง support แบบไหน
