# Alpha Known Limitations M107

## Purpose

Use this document to keep alpha communication honest. Share or summarize these limitations before inviting testers.

## Known Limitations

- AI provider may still be disabled or limited depending on the release state.
- AI can be wrong, incomplete, or unsuitable for a specific farm.
- Weather uses coarse default location unless configured.
- Weather is for planning support, not official emergency warning.
- My Farm and Farm Records are local-first.
- No cloud sync in current V1 alpha.
- No GPS or precise location in current V1 alpha.
- No receipt upload or OCR in current V1 alpha.
- No notifications in current V1 alpha.
- No official government, agronomy, yield, profit, chemical, or accounting guarantee.
- Android/iOS wrapper is not final yet.
- Browser/PWA behavior may differ by phone and browser.

## Alpha Feedback Purpose

The first alpha is for learning:

- whether farmers understand the first screen
- whether Ask AI is useful enough to keep improving
- whether Weather is readable and practical
- whether tools/calculators are easy to use
- whether My Farm/Farm Records feel simple enough
- what blocks real mobile use

## What To Fix First

Fix first:

- broken routes
- unreadable mobile UI
- misleading safety/privacy copy
- Weather production failure
- confusing first-screen navigation
- feedback form/support/privacy blockers

Defer:

- broad new features
- cloud sync
- GPS
- OCR or receipt upload
- notifications
- Android/iOS wrapper until first feedback is reviewed
