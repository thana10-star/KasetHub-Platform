# Weather Risk Threshold Boundary

M78 threshold values are planning placeholders only.

They are intended to help design UI and tests before expert-reviewed agronomy rules exist.

## Planned Signals

- rain chance / precipitation
- wind speed
- humidity
- temperature
- multi-day rain
- stale forecast state

## Boundary Rules

- Threshold labels must include `เบื้องต้น`.
- Thresholds are not official limits.
- Thresholds are not chemical label instructions.
- Thresholds are not legal, insurance, finance, or government scheme advice.
- Thresholds must not recommend products or sponsors.
- Thresholds must not generate exact chemical or fertilizer dose.
- Thresholds must not guarantee yield, profit, safety, or harvest quality.

## Future Review

Before these rules become real agronomy guidance, they need:

- agriculture expert review
- safety review
- source/version metadata
- field applicability notes
- rollback path
- user-facing freshness/version copy

M79 tracks this as rule version metadata and keeps all reviewer sign-offs pending.
